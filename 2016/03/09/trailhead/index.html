Trailhead 教你学习 Salesforce

发表于 2016-03-09

Trailhead 是 Salesforce Doc team 推出的一个互动教程，和 Workbook 类似但更具互动性，地址是

https://developer.salesforce.com/trailhead

Trailhead 的名称

Trail 是小径，Trailhead 是小径的起点。Salesforce 中 Trail 就是通向 “Salesforce 技能树” 的小径，Trailhead 是学习 Salesforce 的起点，在 Trailhead 学习 Salesforce 的人都是 Trailblazer（开拓小径的拓荒者）。

Trail, Module, Project

Trailhead 里有许多 Module 和 Project，它们又由一个个 Unit 组成。一些相关的 Module 组成了一个个 Trail，对应着一个个技能，而Project 则相当于 “项目实战”。

每一个 Unit 都由内容和测验组成，看完了内容，然后完成一组测验，测验包括两种：选择题和实际操作题。实际操作题需要登录 Org，在自己的 Org里完成，然后 Trailhead 检测完成情况。

截止现在为止一共有下面 19个 Trail，包含了 Admin，Developer，Business User 等等

Admin Beginner

Salesforce 配置和自定义

Admin Intermediate

通过拖拽工具来构建应用

Admin Advanced

包含高级的公式、事件监控、Lightning Connect

Developer Beginner

学习简单的 Apex、Visualforce 和数据开发技能

Developer Intermediate

学习移动端开发、软件开发管理、外部系统集成等

Develop with Mobile SDK

使用 Mobile SDK 来构建 iOS，Android 应用

Learn CRM Essentials

理解 Salesforce 和背后的 CRM 理念

Cultivate Equality at Work

这个是输出价值观，不久前新闻说 Salesforce 在企业内倡导男女同工同酬

Manage Fundraising for Nonprofits

专供非营利组织的应用，salesforce.org

Explore with Wave Analytics

Analytics Cloud 应用，在这里可以申请一个包含简单版分析云的开发 Org

Integrate Microsoft Email and Salesforce

学习 Microsoft Outlook 和 Salesforce 的集成

Build Apps as an ISV Partner

ISV Partner Program 的一些工具和方法，原来 ISV Partner 是可以免费申请的

Manage the Salesforce Way

学习 Salesforce Great Manager Model, 成为一个好管理者的策略

Prepare for Salesforce Releases

学习最新的 Salesforce release 功能，做好自己org应用的 release 管理

Navigate the Salesforce Advantage

通过价值、创新技术、生态来了解 Salesforce 相比起竞争者的优势

Sell Lightning Fast with Salesforce

如何使用 Salesforce 让销售卖得更快

Getting Started with Lightning Experience

开始 Lightning 的旅程

Migrate to Lightning Experience

为 Sales team 启用并迁移到 Lightning Experience

Develop for Lightning Experience

使用 Visualforce，Lightning Component，Design System 开发 Lightning 应用，Lightning 的开发重客户端

Points 和 Badges

Trailhead 中每完成一个 Unit 都会获得 Points，如果是选择题，第一次就全答对就有 100分，第二次50分，第三次或更多次才全答对得25分。如果是 实际操作题，只要在 Org 里完成并且通过了 Trailhead 的检测就能得 500分。

另外每完成1个 Module 或者 Project 都可以获得一个 Badge，截至现在，我已经有 78个 Badge。

其他

操作题中的检测验证结果使用了 Tooling API，来验证 Org 中的变更。

除了 Trailhead 外，文档依然是学习 Salesforce和提升技能的必要途径。
