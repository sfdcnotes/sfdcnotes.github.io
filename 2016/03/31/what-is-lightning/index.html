Lightning 到底是什么？

发表于 2016-03-31

这几年 Lightning 火了起来，不过我搞不明白 Lightning 在 Salesforce 中具体指哪些东西，最近在网上搜索了一下，理清了概念。

和 Lightning 这个名字有关的东西有下面这些，我从功能类别把它们分为 数据集成工具，全新的UI，相关开发工具。

数据集成工具

  • Lightning Connect: Lightning Connect 是一个实时数据集成工具，可以在 Force.com 中获取外部数据源的数据，数据传输遵循 OData 规范。现在 Lightning Connect 已经更名为 Salesforce Connect。

全新的 UI

  • Lightning Experience: Lightning Experience 改变了用户界面和用户体验，视觉上更加清爽，跨平台跨屏幕大小视觉一致。

  • Lightning Schema Builder: 可视化的元数据建模工具，可以创建对象、字段、关系。这个就是原来的 Schema Builder，在 Lightning Experience 模式下也可以用，其他没有任何变化。

  • Lightning Process Builder: 这是一个全新的工作流工具，配置界面更加直觉。

  • Lightning Community Builder: Salesforce Community 的建站工具。

组件化开发框架和相关工具

  • Lightning Component Framework: 组件化前端框架，使用 Apex 和 JavaScript 开发可复用的 Lightning Component

  • Lightning App Builder: 点击拖拽的构建工具，管理员可以将开发好的 Lightning Component 组件拼成一个 Lightning Page

  • Lightning Component AppExchange: 开发好的组件可以在 AppExchange 上卖，用户可以在 AppExchange 购买别人开发好的组件

  • Lightning Design System: 用户体验的 style guide 以及一套 CSS UI 框架，符合 Lightning Experience 的视觉体验
