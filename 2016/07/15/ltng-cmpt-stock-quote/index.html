Lightning Component - 股票报价组件

发表于 2016-07-15

股票报价组件由三个松耦合的组件(Loosely Coupled Component)组成。视频中展示了最终这三个独立组件之间的交互。

三个 Component

搜索框 StockQuoteSearchBar.cmp
股票实时报价 StockQuoteLatest.cmp
股票历史k线图 StockQuoteHistoryChart.cmp

一个 Event，用于搜索框和另外两个组件之间的数据交互

StockQuoteSearchKeyChange.evt

Loosely coupled components and communications using Application Event

股价变化 Application Event

在搜索框注册 Event

在实时股价和历史k线图加上 handler，捕获事件中的属性（股票代码）:

Two-way data binding and change tracking

用户输入股票代码后触发 change 事件，随后代码触发股价变化事件

Server-side controller, client-side controller and helper

股价变化事件触发后，历史k线图获取事件中的属性，调用 Yahoo 的 api，获取过去一年的股价
